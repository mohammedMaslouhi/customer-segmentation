# Customer Segmentation

This project focuses on customer segmentation using machine learning techniques.

## Objectives
- Analyze customer behavior
- Segment customers into groups
- Support business decision-making

## Dataset
The dataset contains customer purchase behavior and transaction data.

## Tools
- Python (Pandas, Scikit-learn, Matplotlib)
- K-Means clustering

## Key Steps
- Data preprocessing
- Feature selection
- Clustering (K-Means)
- Visualization of customer segments

## Results
- Identification of distinct customer groups
- Insights for targeted marketing strategies
